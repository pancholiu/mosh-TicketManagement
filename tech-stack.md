# Tech Stack

## Frontend
- **React** with TypeScript
- **Tailwind CSS**
- **React Router**

## Backend
- **Node.js**

## Authentication
- Database sessions (session data stored in PostgreSQL)

## Database
- **PostgreSQL**

## ORM
- **Prisma**

## AI
- **Claude API** (Anthropic)

## Email
- **SendGrid** or **Mailgun** (inbound email parsing via webhook)

## Deployment
- **Docker**
- Cloud provider: AWS, Railway, Fly.io, or equivalent
